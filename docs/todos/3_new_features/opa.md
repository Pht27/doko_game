I want to implement Computer player that automatically plays the first playable card when its its turn, in case I dont get the full 4 players. Lets call that player "Opa".

Opa meldet nie einen Vorbehalt an, Opa sagt nie an. Opa aktiviert keine Sonderkarten. Opa ist immer bereit für das nächste Spiel.

Um Opa einem Spiel hinzuzufügen soll bei einem leeren Seat in einer Lobby ein kleiner Button rechts im Seat erscheinen, der ein kleines Computer Icon zeigt. Jeder Spieler *in der Lobby* kann das drücken und dann wird Opa der Lobby hinzugefügt. Wenn Opa in einem (oder mehreren) Seats ist, taucht ein kleines rotes X auf, mit dem man Opa wieder aus dem Seat entfernen kann.
