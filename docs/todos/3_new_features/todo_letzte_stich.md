Der Ansagen Knopf soll von links nach rechts gehen.

Dafür soll an dessen Stelle ein kleiner Knopf sein, auf den man klicken kann, um nochmal den letzten Stich anzusehen. Falls es noch keinen Stich gab, soll der Knopf nicht angezeigt werden. Der Knopf kann jederzeit betätigt werden (vorausgesetzt er ist da). Wichtig ist, dass hier die Orientation der Karten im angezeigten Stich genauso ist, wie sie auf dem Tisch wäre, also dass die Orientation den Spieler, der die entsprechende Karte gespielt hat impliziert.

Das ganze soll so ein kleines hoverndes Fenster sein, einfach in der Mitte des Screens.

Falls dafür eine backend Abfrage notwendig ist auf einen Endpoint der noch nicht existiert, plane das bite gut durch.
